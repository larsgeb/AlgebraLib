Detailed Description

Functions for AlgebraLib acting on const sparse matrices and vectors.

Author
Lars Gebraad
Date
August 2017

Assorted function which combine 'sparse' algebra classes within the library to execute typical linear algebra functions. Most of the inputs here are const, and the functions won't alter the input.
