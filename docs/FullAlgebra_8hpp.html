Detailed Description

Functions for AlgebraLib acting on const full matrices and vectors.

Author
Lars Gebraad
Date

Assorted function which combine full algebra classes within the library to execute typical linear algebra functions. Most of the inputs here are const, and the functions won't alter the input.
